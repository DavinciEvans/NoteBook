# 数字逻辑的基本定律

AB 与

A + B 或

交换律、结合律、分配律依然适用

## 0-1律

A + 0 = A

A + 1 = 1

A · 0 = 0

A · 1 = A

## 互补律

A + A' = 1

A · A' = 0

## 其余定理

幂等律

- A + A = A
- A · A = A

---

吸收律

- A + AB = A
- A(A+B) = A

---

- A + A'B = A + B
- A(A' + B) = AB

---

双重否定

- A'' = A

---

摩根律

- (A + B)' = A' · B'
- (AB)' = A' + B'
