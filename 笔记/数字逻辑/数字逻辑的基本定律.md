# 数字逻辑的基本定律

AB 与

A + B 或

交换律、结合律、分配律依然适用

## 0-1律

A + 0 = A

A + 1 = 1

A · 0 = 0

A · 1 = A

## 互补律

A + A' = 1

A · A' = 0

## 其余定理

幂等律

- A + A = A
- A · A = A

---

吸收律

- A + AB = A
- A(A+B) = A

---

- A + A'B = A + B
- A(A' + B) = AB

---

双重否定

- A'' = A

---

摩根律

- (A + B)' = A' · B'
- (AB)' = A' + B'

---

合并定则

- AB + AB' = A
- (A + B) · (A + B') = A

---

- AB + A'C + BC = AB + A'C
- (A + B)(A' + C)(B + C) = (A + B)(A' + C)

## 反演规则

所有 0 变 1，1 变 0，T 变 F，F 变 T。与变或，或变与，原变量变成反变量，反变量变成原变量。即可以得到原函数 F 的反函数 F'

## 对偶规则

在反演规则中去掉变量的变换
