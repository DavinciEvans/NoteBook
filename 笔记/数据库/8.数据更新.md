# 数据更新

- 插入元组
- 插入查询结果（即插入多个元组）

语句格式：

```SQL
INSERT INTO <表名> [(<属性列1>[,<属性列2 >…)]
VALUES (<常量1> [,<常量2>]… );
```

关于 INTO 子句：

- 指定要插入数据的表名及属性列
- 属性列的顺序可与表定义中的顺序不一致
- 没有指定属性列：表示要插入的是一条完整的元组，且
属性列属性与表定义中的顺序一致
- 指定部分属性列：插入的元组在其余属性列上取空值

VALUE 子句必须与 INTO 匹配

即值的数量与类型相互匹配

🌰：

```SQL
INSERT INTO Student (Sno,Sname,Ssex,Sdept,Sage)
VALUES ('201215128','陈冬','男','IS',18);
```

插入选课记录：

```SQL
INSERT INTO SC(Sno,Cno)
 VALUES ('201215128 ',' 1 ');
```

这个命令与下面命令等效：

```SQL
INSERT INTO SC
 VALUES (' 201215128 ',' 1 ',NULL);
```
